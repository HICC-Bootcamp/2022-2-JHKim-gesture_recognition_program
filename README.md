# 2022-2-JHKim-gesture_recognition_program



### 1. 프로그램 소개

사용자가 특정 동작을 컴퓨터에 학습시켜

특정 동작을 수행할 때 유용한 특정 기능을 실행하는 프로그램

동작을 학습시키기 위해 LSTM사용
